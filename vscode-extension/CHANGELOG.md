# Change Log

All notable changes to the "forma-language" extension will be documented in this file.

## [0.1.0] - 2025-12-26

### Added
- Initial release
- Syntax highlighting for `.forma` and `.fml` files
- Language Server Protocol integration
- Support for code completion
- Diagnostic support (errors and warnings)
- Symbol navigation
- Configurable language server path
- Status bar indicator
- Restart language server command

### Features
- Comments (line and block)
- Keywords (class, property, event, method, enum, when, preview, etc.)
- String literals with escape sequences
- Numeric literals (integers, floats, hex)
- Type annotations
- Operators and punctuation
- Auto-closing pairs for brackets, quotes
- Folding regions
- Indentation rules
